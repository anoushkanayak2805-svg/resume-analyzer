# AI Resume Analyzer

## 🚀 Features
- Resume Parsing (NLP)
- Job Matching (TF-IDF + Cosine Similarity)
- ATS Score Calculation
- AI Suggestions
- MongoDB Integration

## 🛠️ Tech Stack
- Python (Flask)
- React
- MongoDB
- spaCy
- scikit-learn

## ▶️ How to Run
cd backend
python app.py
